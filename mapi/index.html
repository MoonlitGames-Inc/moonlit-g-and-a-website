Ecosystem

Integration with Moonlit Games

Moonlit APIs is the backbone connecting developers to the Moonlit Games ecosystem.

01

Game Data Pipeline

Moonlit APIs acts as the data layer between Moonlit Games titles and third-party applications. Access real-time game state, player progress, and in-game events from titles like Chill Gorilla.

02

Authentication & Accounts

Unified player identity across all Moonlit titles. Single sign-on, profile management, and cross-game progression data powered by the APIs layer.

03

Community & Content Tools

Build community tools, stat trackers, leaderboards, and content platforms using our open API endpoints. Everything connects back to the Moonlit Games ecosystem.
